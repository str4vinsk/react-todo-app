# react-todo-app

Simple todo app created with React JS for learning purposes

![Screenshot](./Screenshot.png)

---

## 💾 Installing 

Git clone the repository

```
git clone https://github.com/str4vinsk/react-todo-app
```

Install the dependencies

```
yarn install
```

And run it!

```
yarn start
```
